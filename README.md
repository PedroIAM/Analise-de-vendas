# Analise-de-vendas
Projeto desenvolvido com o objetivo de praticar integração entre PostgreSQL, Python e Pandas, realizando consultas SQL e análises de dados reais de vendas.
## 🚀 Objetivo do Projeto

O principal objetivo deste projeto é aplicar conceitos de:

- Integração entre banco de dados e Python
- Manipulação de dados com Pandas
- Consultas SQL (SELECT, WHERE, ORDER BY, etc.)
- Análise exploratória de dados (EDA)
- Geração de insights a partir de dados de vendas

---

## 🛠️ Tecnologias Utilizadas

- Python 🐍
- PostgreSQL 🐘
- Pandas 📊
- Matplotlib 📈 
- SQL
- ## 📈 O que foi analisado

Neste projeto, foram realizadas análises como:

- Produtos mais vendidos
- Faturamento total da loja
- Vendas por período
- Produtos com menor desempenho
- Identificação de padrões de consumo




## 📌 Observação

Este projeto utiliza uma **base de dados fictícia** de uma loja de informática, criada apenas para fins de estudo e prática.
